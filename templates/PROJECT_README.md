## 🚀 Usage / نحوه‌ی اجرا

### Quick Run
```bash
# English
python main.py --mode=simulate --duration=3600

# فارسی (اگر اسکریپت از آرگومان فارسی پشتیبانی می‌کنه)
python main.py --mode=شبیه‌سازی --زمان=3600
```

### Configuration
Edit `config.yaml` or `settings.m` to customize:
```yaml
orbit:
  semi_major_axis: 7000e3  # meters
  inclination: 98.6        # degrees (Sun-synchronous)
  epoch: "2026-01-01T00:00:00Z"
```

### Expected Output
- 📊 `output/trajectory.csv` — Orbital state vectors over time
- 🖼️ `output/plot_3d.png` — 3D visualization of the orbit
- 📝 `output/report.txt` — Summary statistics and convergence info
