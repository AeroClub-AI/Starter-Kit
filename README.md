## 🌍 English
Welcome to **AeroClub-AI Starter-Kit** — an open-source, student-driven collection of aerospace and AI projects, datasets, and tutorials built for Iranian researchers. Whether you're working on CFD, flight dynamics, satellite ML, or avionics, this repository gives you a clean starting point, offline-friendly workflows, and direct access to global open-source standards.  
🌐 *No constant internet? No problem. Clone it once, work locally, sync later.*

## 🇮🇷 فارسی
به **کیت شروع پروژه‌های هوافضا و هوش مصنوعی AeroClub-AI** خوش آمدید. این مخزن برای دانشجویان و پژوهشگرانی طراحی شده که می‌خوان بدون وابستگی به اینترنت پایدار، روی پروژه‌های واقعی هوافضا و سیستم‌های هوشمند کار کنن. از شبیه‌سازی جریان و دینامیک پرواز تا یادگیری ماشین فضایی و آویونیک، اینجا همه چیز آماده‌ی اجرا، مستند و قابل توسعه‌ست.  
📶 *کار آفلاین؟ بله. یه بار کلون کن، ماه‌ها کار کن، هر وقت تونستی سینک کن.*

## 🚀 Quick Start / شروع سریع
```bash
# 1. Clone the repository
git clone https://github.com/AeroClub-AI/Starter-Kit.git
cd Starter-Kit

# 2. Check example requirements (Python example)
cd examples/orbit-prediction
pip install -r requirements.txt

# 3. Run & explore
python main.py
```
> 💡 **Tip:** For MATLAB/C++ examples, open the respective folders and follow the `README.md` inside each.
